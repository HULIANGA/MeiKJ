一、目录说明
1.build
  webpack配置文件

2.dist
  编译目标目录

3.src
  所有的本地项目代码

  src/app
    对应每个html的vue组件

  src/assets
    css和图片

  src/components
    组件

  src/entrys
    对应每个app组件的入口js

  src/html
    页面模板

  src/js
    自己封装的js方法

  src/libs
    第三方js库和第三方json数据

4.static
  临时使用的图片和json数据文件，不会打包到发布项目中

二、配置文件
.babelrc                babel配置文件
.editorconfig           sublime编辑配置文件 https://packagecontrol.io/packages/EditorConfig
.eslintrc               eslint配置文件
package.json            npm配置文件

三、how to start
install dependencies
npm install

clean dist
npm run clean

build for production with minification
npm run build

serve with hot reload at localhost:8080
npm run dev

serve with hot reload at port:8080
npm run dev-port

run unit tests
npm test
