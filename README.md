# BIGAKA-VUE
vue框架
一、目录说明
apps  单个页面vue组件
build 项目构建指令
components  拆分的vue组件
css  css文件
dist  构建导出目录
entrys  html入口js文件
html  html文件
img  图片文件
js  js文件
--libs  js插件库
--services  封装js方法
json  静态json数据
node_modules  npm包

二、配置文件
.babelrc  babel配置文件
.editorconfig  sublime编辑配置文件 https://packagecontrol.io/packages/EditorConfig
.eslintrc  eslint配置文件
package.json  npm配置文件

三、how to start
``` bash
install dependencies
npm install

serve with hot reload at localhost:8080
npm run dev

build for production with minification
npm run build

run unit tests
npm test
```
